# Фоновые изображения

Эта папка предназначена для хранения фоновых изображений сайта.

## Необходимые изображения:

1. **bg-pattern.svg** - Фоновый паттерн для всей страницы
2. **hero-bg.svg** - Фоновое изображение для Hero секции
3. **features-bg.svg** - Фоновое изображение для Features секции
4. **usecases-bg.svg** - Фоновое изображение для UseCases секции
5. **pricing-bg.svg** - Фоновое изображение для Pricing секции

## Формат изображений:

- Рекомендуется использовать SVG для масштабируемости
- Также можно использовать PNG/JPG с высоким разрешением
- Размеры: минимум 1920x1080px для фоновых изображений

## Как добавить изображения:

1. Скопируйте изображения с сайта https://tele-radar-pulse.lovable.app/
2. Сохраните их в эту папку с указанными именами
3. Изображения автоматически подхватятся компонентами

## Альтернатива:

Если изображения недоступны, сайт будет работать с градиентными фонами и паттернами по умолчанию.
