# Флаги стран

Поместите сюда изображения флагов для языков:

- `RU.webp` - флаг России (для русского языка)
- `GB.webp` - флаг Великобритании (для английского языка)  
- `ES.webp` - флаг Испании (для испанского языка)

Рекомендуемый размер: 20x15px или больше (будет масштабироваться).
Формат: WebP, PNG или SVG.

Вы можете скачать флаги с:
- https://flagcdn.com/
- https://www.countryflags.com/
- Или использовать любые другие источники
